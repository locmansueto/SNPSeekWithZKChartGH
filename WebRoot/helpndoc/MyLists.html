       The My Lists page serves as the users workspace. It displays the lists that the user have created either from query results or by manual edit in this page.



To create a Variety List:


    1. Click the Create button
    2. Type in or paste the variety names, IRIS IDs or IRGC accessions. One entry for every line. If several names or IDs or accessions refer to the same entity, only one is created.
    3. Click the Save button, or Cancel


To delete a Variety List:

    1. Select the list name
    2. Click the Delete button.
